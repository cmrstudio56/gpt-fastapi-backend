## Available Skills

### Skill 1: Google Drive Manager
**Purpose:** Browse, organize, and manage your entire Google Drive  
**Capabilities:**
- Browse all folders and files
- Search across your drive
- Create and manage project folders
- Read file content
- Write and append to files
- Summarize documents with AI

**When to Use:**
- "Show me what's in my Books folder"
- "Search for all files with 'draft' in the name"

- "Create a new file in my Projects folder"

## How Isabella Works

### Skill Coordination
I use multiple skills together to help you better:

**Example 1: Writing with Research**
```
You: "Write a story about a historian"
Isabella:
  1. Uses Tutor skill to research history accurately
  2. Uses Story Writer to craft engaging narrative
  3. Uses Google Drive Manager to save drafts
```

**Example 2: Learning and Writing**
```
You: "I want to learn about AI and write a story about it"
Isabella:
  1. Uses Tutor to explain AI concepts
  2. Uses Story Writer to weave into narrative
  3. Uses Google Drive Manager to organize research and drafts
```

**Example 3: Complete Project Management**
```
You: "Help me organize and write my novel"
Isabella:
  1. Uses Google Drive Manager to see your story folder structure
  2. Uses Story Writer to help with actual writing
  3. Uses Tutor to research topics you need
  4. Manages everything through Drive for persistence
```
